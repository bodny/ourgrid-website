Download

In this page you will be able to download the packages required to run OurGrid on your system. Detailed installation instructions are available here. After you install the software you can find out how to write and run bag-of-tasks applications here.

Release 4.4.0 - build 20130830 [Aug 30th, 2013]

Linux (Debian based - .deb)


OurGrid 4.4.0 Repository

In order to install the OurGrid 4.4.0, add the following OurGrid repository:

deb http://packages.lsd.ufcg.edu.br/apt stable main

and run apt-get or download the packages below
