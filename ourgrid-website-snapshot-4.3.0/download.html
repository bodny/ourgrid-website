Release 4.3.0 - build 20120530 [May 30th, 2012]

OurGrid 4.3.0 Repository

In the OurGrid version 4.3.0, the installation process was simplified and every OurGrid component can be installed by the OS package manager like Synaptic. In order to install any OurGrid component, you must add the OurGrid repository to your source list.

Linux (Debian based - .deb)


In orde to install the OurGrid, add the following OurGrid repository:

deb http://packages.lsd.ufcg.edu.br/apt stable main

or download the packages below


OurGrid Common ourgrid-common_4.3.0-build-20120530_all.deb
OurGrid Peer ourgrid-peer_4.3.0-build-20120530_all.deb
OurGrid Worker without virtualization ourgrid-worker_4.3.0-build-20120530_all.deb
OurGrid Worker VServer ourgrid-worker-vserver_4.3.0-build-20120530_all.deb
OurGrid Worker Virtual Box ourgrid-worker-vbox_4.3.0-build-20120530_all.deb
OurGrid Broker ourgrid-broker_4.3.0-build-20120530_all.deb
OurGrid Discovery Service ourgrid-ds_4.3.0-build-20120530_all.deb
